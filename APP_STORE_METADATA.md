## 应用描述

### 中文 (4000字符限制)

Nanomouse 是一款基于 RIME 引擎的开源跨平台输入法，让拼音输入更快、更顺手。

【核心特色】
• 后鼻音简化：用 nn 代替 ng，如 dann → 当，少按一键
• 键位优化：用 vn/vnn 代替 uan/uang，如 gvn → 关，手指更舒适
• 不改变原有习惯，只优化高频操作

【丰富功能】
• 按键气泡与触感反馈，体验如原生键盘
• 长按字母键弹出扩展字符菜单
• 长按数字键快速输入数字，无需切换
• 长按地球键一键切换中/日/英
• 自动读取 iOS 系统文本替换

【多方案支持】
• 雾凇拼音 - 现代化拼音词库
• 地球拼音 - 带声调标注
• 日语罗马字 - 日文输入
• 笔画输入 - 手写习惯用户
• 更多方案持续添加中

【隐私保护】
• 开源代码，公开透明
• 不收集任何输入内容
• 不使用任何第三方分析服务
• 联网仅用于你的 iCloud 同步

### 英文 (4000字符限制)

Nanomouse is an open-source, cross-platform input method based on the RIME engine, designed to make Pinyin typing faster and more comfortable.

【Key Features】
• Nasal sound shortcuts: Use "nn" instead of "ng" (e.g., dann → 当)
• Optimized key positions: Use vn/vnn instead of uan/uang (e.g., gvn → 关)
• No learning curve - just smoother high-frequency typing

【Rich Experience】
• Key bubbles and haptic feedback, feels like the native keyboard
• Long-press any key for extended character menu
• Long-press 123 for quick number input without switching
• Long-press globe key for instant CN/JP/EN switching
• Auto-syncs with iOS Text Replacement

【Multiple Schemas】
• Rime Ice - Modern Pinyin dictionary
• Terra Pinyin - With tone marks
• Japanese Romaji - Japanese input
• Stroke Input - For handwriting-style users
• More schemas coming soon

【Privacy First】
• Open-source and transparent
• No input data collection
• No third-party analytics
• Network access only for your iCloud sync
