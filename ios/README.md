# iOS 仓输入法配置说明

## 安装步骤

### 1. 安装仓输入法

在 App Store 搜索"仓输入法"（Hamster）并安装。

### 2. 导入配置

**方法一：通过文件 App**
1. 将 `hamster` 文件夹中的所有 `.yaml` 文件发送到 iPhone
2. 打开"文件"App，找到这些文件
3. 选中文件 → 分享 → 用仓输入法打开

**方法二：通过 iCloud 同步**
1. 在 Mac/Windows 上将 `.yaml` 文件复制到 iCloud Drive
2. 在 iPhone 上打开仓输入法 → 设置 → 同步
3. 选择 iCloud 同步目录

### 3. 启用键盘

1. 打开 iPhone 设置 → 通用 → 键盘 → 键盘
2. 添加新键盘 → 选择"仓输入法"
3. 点击仓输入法 → 开启"允许完全访问"

### 4. 测试

在任意输入框切换到仓输入法，输入 `dann` 应该能看到"当"等候选词。

## 常见问题

**Q: 导入后没有生效？**
A: 在仓输入法设置中点击"重新部署"。

**Q: 找不到导入选项？**
A: 确保使用的是最新版仓输入法（支持配置导入功能）。
